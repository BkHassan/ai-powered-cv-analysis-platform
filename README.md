# Ai_CV

A platform for analyzing and processing CVs using AI, built with Nest.js, Next.js, and ChromaDB. It allows admins to upload CVs, assign them to departments (e.g., Marketing, Development), generate skill-based tests, and manage candidate communication via email.

## Tech Stack
- **Frontend**: Next.js, React, Tailwind CSS
- **Backend**: Nest.js, TypeScript
- **Database**: ChromaDB (vector database)
- **AI**: OpenAI API (Chat-4)
- **Email**: SendGrid
- **Auth**: JWT
- **Package Manager**: pnpm
- **Deployment**: Docker, docker-compose
